Welcome to 221g Pages.

The idea behind the project is to develop a smart phone application for getting visa US status. The application also provides a wiki and a forum for users to share the experiences and knowledge. The main core of the application is build with sencha touch an open source java script mobile build platform to develop fast mobile applications that can be compiled for multiple plat forms . All of you who have basic javascript/php or sencha touch experience are requested to join in this project . You can contact me @ rakeshr.21@gmail.com

Project Specifications

We are using sencha touch for the application development,for the server side scripting php is the preferred choice. Database is MySQL.

Inspiration to do the project

The idea behind the project is to have an iPhone,android and windows mobile application for all US visa seekers. Where people can post their experiences and concerns about the visa process. Currently there are no smart phone applications for this process and hence, it gave me the inspiration to come up with one. Also the process of waiting for a status change once you are put in an administrative processing is tedious. The most preferable solution is a mobile application.
